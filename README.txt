Petunjuk Pemasangan (Jika Laptop Belum Terinstal Apa-apa):
----------------------------------------------------------
1. **Download Python 3.x**
   - Buka: https://www.python.org/downloads/
   - Klik tombol "Download Python 3.x.x" (pilih versi terbaru).
   - Jalankan file installer yang sudah di-download.
   - **PENTING:** Saat jendela instalasi muncul, CENTANG "Add Python to PATH" di bagian bawah sebelum klik "Install Now".
   - Tunggu proses instalasi selesai, lalu klik "Close".

2. **Cek Python dan pip**
   - Buka Command Prompt (CMD):
     - Tekan tombol `Windows + R`, ketik `cmd`, lalu tekan ENTER.
   - Ketik perintah berikut lalu tekan ENTER:
     ```
     python --version
     ```
     - Jika muncul versi Python, berarti Python sudah terpasang.
   - Cek pip dengan perintah:
     ```
     pip --version
     ```
     - Jika muncul versi pip, berarti pip sudah terpasang.
   - **Jika pip/python tidak dikenali:**
     - Restart komputer, lalu ulangi langkah di atas.
     - Pastikan saat instalasi Python, "Add Python to PATH" sudah dicentang.

3. **Install library yang dibutuhkan**
   - Di CMD, ketik perintah berikut satu baris lalu tekan ENTER:
     ```
     pip install pyautogui numpy colorama keyboard
     ```
   - Tunggu sampai semua proses selesai (akan muncul tulisan "Successfully installed ...").

4. **Siapkan folder gambar**
   - Pastikan semua file gambar tombol (misal: ok_button_day.png, spin_button.png, dst) ada di dalam folder `img`.
   - Folder `img` harus berada di satu folder dengan file `tes.py` dan `run_tes.bat`.

5. **Jalankan script**
   - Klik 2x file `run_tes.bat` untuk menjalankan script.
   - Atau buka CMD, masuk ke folder script, lalu jalankan:
     ```
     python tes.py
     ```

Petunjuk Penggunaan:
--------------------
1. Pastikan kamu sudah install Python 3.x di komputer.
2. Pastikan semua file gambar tombol ada di dalam folder 'img' (satu folder dengan tes.py).
3. Untuk menjalankan script, cukup klik 2x file:
   run_tes.bat
   (atau jalankan di terminal/cmd: python tes.py)
